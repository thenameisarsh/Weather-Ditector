# 🌦️ Weather Detector

A simple and responsive Weather Detector web application that provides real-time weather information for any city using the OpenWeather API.

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Real-time temperature display
- 💧 Humidity information
- 🌬️ Wind speed details
- ☁️ Weather condition with icons
- 📱 Responsive design
- ⚠️ Error handling for invalid city names

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeather API

## ⚙️ Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/weather-detector.git
```

2. Navigate to the project folder:
```bash
cd weather-detector
```

3. Open `index.html` in your browser.

## 🔑 API Setup

1. Create a free account at OpenWeather.
2. Generate your API key.
3. Replace the placeholder API key in `script.js`:

```javascript
const API_KEY = "YOUR_API_KEY";
```
